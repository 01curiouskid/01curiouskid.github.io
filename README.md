# Adarsh Gupta | Portfolio

Welcome to my personal portfolio repository! This website showcases my projects, experience, and education as a Software Development Engineer and Data Science enthusiast.

## Features
- **Dynamic Terminal UI:** A custom macOS-style terminal hero section.
- **Projects Showcase:** Isotope-filtered grid of my ML/AI and Software Engineering projects.
- **Responsive Design:** Fully responsive layout built on Bootstrap.
- **Smooth Scrolling:** Native CSS smooth scrolling.

## Development
To run this project locally, simply start a static server in this directory:
```bash
python3 -m http.server 8080
```
Then navigate to `http://localhost:8080`.
